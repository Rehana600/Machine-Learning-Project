# Machine Learning Mini Projects

This repository contains four simple machine learning projects that I created as part of my learning. These projects are based on real-world problems and show how machine learning can be applied in different areas.

## Projects Included

### 1. Customer Churn Prediction
This project predicts whether a customer will leave a service or stay. It helps businesses understand customer behavior and improve retention.

### 2. Sales Forecasting
This project predicts future sales based on past data. It helps in understanding trends and planning ahead.

### 3. Credit Risk Prediction
This project predicts whether a customer is a good or bad credit risk. It is useful for financial decision-making.

### 4. HR Attrition Prediction
This project predicts whether an employee is likely to leave the company. It helps in analyzing employee turnover.

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib

## Workflow
Each project follows these steps:
- Data loading
- Data cleaning
- Handling missing values
- Converting categorical data into numeric form
- Splitting data into training and testing sets
- Training machine learning models
- Evaluating results
- Creating graphs for visualization

## Models Used
- Random Forest Classifier
- Decision Tree Classifier
- Linear Regression

## How to Run

Install required libraries:
pip install pandas numpy scikit-learn matplotlib

Run the project files:
python churn.py
python sales.py
python credit.py
python hr.py
